Anti-AI Writing Taiwan
降低 AI 寫作痕跡，保留作者原本的語氣與風格。
這是一套專為繁體中文設計的文字編輯 Skill，目的是降低 AI 常見的寫作痕跡，同時保留作者原本的觀點、語氣與寫作風格。
本 Skill 以台灣讀者的閱讀習慣為主要設計對象，適合搭配 ChatGPT、Claude、Gemini 等大型語言模型使用。

專案特色
與其重新改寫文章，本 Skill 更像一位資深文字編輯。
它不追求把文章修得完美，而是協助 AI 以更自然、更符合真人寫作習慣的方式表達內容：

保留作者原本的觀點與風格
降低常見的 AI 寫作痕跡
改善文章節奏與閱讀流暢度
避免制式化、教科書式或翻譯腔的表達
使用符合台灣讀者閱讀習慣的用字與語感


適用情境
適合：

部落格文章
Facebook 貼文
LinkedIn 文章
Medium 文章
電子報
行銷文案
書籍草稿

不建議直接用於：

醫療文件

程式碼


使用方式
複製 SKILL.md 的內容，貼到 AI 工具的 System Prompt、專案指令（Project Instructions），或做為對話的第一則指令，再提供需要修改的文章。
Skill 會依照以下流程進行：

通讀全文，找出最明顯的 AI 痕跡。
優先調整整體節奏，再修改局部細節。
保留作者原意，只修改真正需要修改的地方。

設計理念
好的文字編輯，不是把文章全部重寫。
真正重要的是保留作者的想法、語氣與個性，只調整那些會影響閱讀體驗的 AI 寫作痕跡。
自然，比工整更重要。
